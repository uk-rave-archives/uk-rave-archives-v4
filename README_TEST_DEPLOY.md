# GitHub Pages Test Deployment

This package is prepared for a **separate test repository**. Do not upload it over the live archive until the test version has been approved.

## Upload

1. Create a new GitHub repository, for example `uk-rave-archives-v4-test`.
2. Extract this ZIP on your computer.
3. Upload the **contents inside the extracted folder** to the root of the new repository.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, select **Deploy from a branch**.
7. Select `main` and `/ (root)`, then press **Save**.
8. Wait for the Pages deployment to finish.

The address will be similar to:

`https://uk-rave-archives.github.io/uk-rave-archives-v4-test/`
